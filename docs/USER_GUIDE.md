# GeekClock 使用说明书

> 一款贴心、不打扰的桌面闹钟软件

## 一、初次使用

### 1.1 安装

**方式 A：直接使用打包版本（推荐）**

收到 `GeekClock.exe` 后，把它放到任意文件夹（建议新建一个专门文件夹，如 `D:\Apps\GeekClock\`），双击运行即可。

> 程序会在 exe 同级目录自动创建 `logs/` 文件夹用于存放日志。
> 闹钟配置会自动保存到 `C:\Users\<你的用户名>\AppData\Roaming\GeekClock\alarms.json`，重装/升级不会丢失。

**方式 B：源码运行**

```bash
# 安装 Python 3.12.10 + uv 后运行：
uv sync
uv run python main.py
```

### 1.2 第一次启动

启动后你会看到：
- **主窗口**：闹钟列表（首次启动可能为空）
- **任务栏右下角的托盘图标**：GeekClock 在后台运行的标志

主窗口的特点：
- **关闭按钮（×）不会真正退出程序**，只是隐藏到托盘
- 想真正退出，需要**右键托盘图标 → 退出GeekClock**

## 二、闹钟功能

### 2.1 三种闹钟类型

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| **间隔重复** | 每隔 N 秒/分钟/小时响一次 | 喝水、休息眼睛、站起来活动 |
| **固定时刻** | 每天/每周指定时间响 | 起床、吃药、晨会 |
| **单次** | 指定日期时间响一次 | 临时任务、约会提醒 |

### 2.2 新建闹钟

**方法一**：主窗口右上角点 **+ 新建闹钟**
**方法二**：右键托盘图标 → **新建闹钟**

填写界面：

1. **名称**：闹钟的名字（如"喝水提醒"）
2. **分组**：将闹钟归类（如"健康"、"工作"），同名闹钟会自动归入同一组
3. **触发方式**：三选一
   - 间隔重复：每隔多久 + 活动时段 + 工作日选择
   - 固定时刻：具体几点 + 星期几
   - 单次：具体哪一天哪几点
4. **提示文本**：通知里显示的内容
5. **提示音**：选择 sounds/ 目录里的音频，或浏览自定义音频
6. **音量**：0-100%
7. **渐强淡入**：勾选后音频从静音渐增到目标音量
8. **允许延后**：勾选后通知里有"延后"按钮

### 2.3 闹钟分组管理

闹钟按分组自动归类显示。每个分组可以：
- **点击分组标题**：展开/折叠该分组下的所有闹钟
- **点击"全部暂停/全部启用"按钮**（分组标题右侧）：一键控制该分组所有闹钟

折叠的分组不占空间，方便管理大量闹钟。

### 2.4 管理闹钟

每条闹钟卡片有三个操作：
- **编辑**：修改名称、时间、音频等
- **✕**：删除闹钟（会确认）
- **开关**：临时启用/禁用，不删除

颜色编码：
- 🟢 绿色图标 = 间隔重复
- 🟣 紫色图标 = 固定时刻
- 🟠 橙色图标 = 单次

### 2.5 闹钟触发时

**音频播放** + **屏幕右下角弹出通知**

通知特点：
- **不会抢键盘焦点**——你正在打字、玩游戏不会被打断
- 8 秒后自动关闭
- 鼠标悬停在通知上时，倒计时暂停

通知上的按钮：
- **延后…** → 弹出延后选项窗口
- **知道了** → 关闭通知，闹钟下次按计划再响

### 2.6 延后功能

点"延后…"后，屏幕中央弹出选项窗口：

**快捷预设**（可在设置里自定义）
- 默认 5 分钟 / 15 分钟 / 30 分钟 / 1 小时

**自定义时长**
- 输入数字 + 选择"分钟后"或"小时后"

**特殊时间**
- 明天上午（默认 09:00，可设置里改）
- 今天稍后（默认 2 小时后，可设置里改）
- 选择具体时间（弹出日期时间选择器）

### 2.7 全部暂停 / 全部启用

主窗口右上角"全部暂停"按钮，一键暂停所有闹钟。
也可以在**右键托盘图标 → 全部暂停 / 全部启用**操作。

## 三、勿扰时段

不希望深夜被吵醒？设置勿扰时段：

1. 打开**设置**（主窗口"⚙ 设置"按钮 / 托盘菜单"设置…"）
2. 勾选 **启用勿扰时段**
3. 设置开始和结束时间（如 22:00 - 08:00）
4. 保存

勿扰时段内**所有闹钟都不会触发**（包括音频和通知都不响）。

## 四、音响保活

防止 Windows 音频设备因长时间不活动而进入休眠，导致闹钟不响。

### 4.1 设置

点击主窗口**左下角的音响图标（📢）**，弹出设置窗口：

- **启用音响保活**：勾选后定期激活音频设备
- **激活间隔**：每隔多少分钟激活一次（1-60 分钟）
- **激活音量**：建议设为 0%（静音激活），不会听到任何声音
- **激活时段**：仅在此时段内激活，如 00:00 - 23:59（全天激活）

鼠标悬停在音响图标上会显示提示"音响保活设置"。图标颜色表示状态：绿色=已启用，灰色=已禁用。

### 4.2 快速开关

- **点击左下角音响图标** → 打开完整设置
- **右键托盘菜单** → "启用音响保活 / 禁用音响保活" → 快速切换

## 五、桌面悬浮时钟

### 5.1 显示/隐藏

**右键托盘图标 → 显示桌面时钟**

### 5.2 两种形态

**时钟模式**（默认）：白色卡片，显示完整时间（含秒）+ 日期 + 下一个闹钟预告

**小图标模式**：圆形图标 + 角标数字（数字表示启用的闹钟数）

切换：右键悬浮时钟 → "切换为小图标 / 切换为时钟"

### 5.3 移动位置

**用鼠标拖拽**即可。位置会自动记住。

### 5.4 调整透明度

右键悬浮时钟 → 拖动**透明度滑动条**（0% - 100%）

### 5.5 锁定位置（穿透点击）

右键 → **锁定位置**

锁定后鼠标点击会直接穿透到下面的窗口。**怎么解锁**：右键托盘图标 → "解锁悬浮时钟"

### 5.6 双击操作

**双击悬浮时钟** = 打开主窗口

## 六、计时器 / 倒计时

### 6.1 显示/隐藏

**右键托盘图标 → 显示倒计时**

### 6.2 两种模式

**倒计时**（默认）：从设定时长递减到 0，到时响铃 + 弹通知
**秒表**：从 0 开始累加，可暂停可重置

切换：右键 → "切换为秒表 / 切换为倒计时"

### 6.3 操作方式

| 操作 | 倒计时模式 | 秒表模式 |
|------|----------|---------|
| 单击 | 开始 / 暂停 | 开始 / 暂停 |
| 双击 | 打开设置（改时长） | 重置归零 |
| 拖拽 | 移动位置 | 移动位置 |
| 右键 | 菜单 | 菜单 |

### 6.4 倒计时设置

双击悬浮窗或右键 → "倒计时设置…"：
- 时长：小时 + 分钟 + 秒（快捷预设：5/15/25/45/60 分钟）
- 提示音 + 试听
- 音量

## 七、系统托盘

### 7.1 托盘图标位置

Windows 任务栏最右侧的图标区域。Windows 11 默认隐藏不常用图标，点"^"展开按钮可以看到。

### 7.2 鼠标操作

- **悬停**：显示工具提示（如"3 / 5 个闹钟已启用"）
- **单击 / 双击**：打开主窗口
- **右键**：菜单

### 7.3 右键菜单全览

```
显示主窗口
设置…
新建闹钟
─────────────
全部暂停
全部启用
启用音响保活（/禁用音响保活）
显示桌面时钟（/隐藏桌面时钟）
解锁悬浮时钟（仅锁定时显示）
显示倒计时（/隐藏倒计时）
─────────────
退出GeekClock
```

## 八、全局设置

主窗口"⚙ 设置"或托盘菜单"设置…"。

### 8.1 勿扰时段

详见第三章。

### 8.2 通知设置

- **延后快捷预设**：通知里"延后"对话框的快捷按钮，用逗号分隔，最多 4 个（如 `10,30,60,120`）

- **「今天稍后」延后**：默认 2 小时

- **开机自启动**：勾选后下次开机自动启动 GeekClock，最小化到托盘运行

## 十、常见问题

### 10.1 使用问题

**Q：双开了 GeekClock 怎么办？**
A：不会真的双开。第二次启动会自动激活第一个实例的窗口然后退出。

**Q：关闭主窗口程序就消失了？**
A：不是消失，是隐藏到托盘了。看任务栏右下角图标。想真正退出：右键托盘 → 退出。

**Q：闹钟到点没响？**
A：排查顺序：
1. 主窗口看那个闹钟的开关是不是绿色（启用状态）
2. 当前时间是不是在闹钟的活动时段内
3. 当前是不是勿扰时段
4. 当前是星期几，闹钟是否覆盖

**Q：听不到声音但通知出现了？**
A：检查：编辑闹钟看 audio 字段、试听按钮、系统音量、应用音量混音器。

**Q：弹通知时光标会跳走吗？**
A：不会。通知设计上不抢键盘焦点。

**Q：悬浮时钟锁定后右键也没反应了？**
A：那是穿透点击的设计。从**右键托盘图标 → 解锁悬浮时钟**来解锁。

**Q：音响保活是什么？需要开吗？**
A：定期播放一段静默音频来保持音频设备活跃。如果你发现闹钟有时不响（特别是电脑闲置一段时间后），建议开启。建议设 0% 音量（完全静音），不会影响正常使用。

### 10.2 数据和备份

**Q：我的闹钟数据存在哪里？**
A：
- **打包版**：`C:\Users\<你的用户名>\AppData\Roaming\GeekClock\alarms.json`
- **源码版**：项目目录下的 `alarms.json`

**Q：怎么备份我的闹钟？**
A：复制上述 `alarms.json` 文件即可。

**Q：升级后闹钟会丢吗？**
A：**不会**。配置在 `%APPDATA%\GeekClock\` 目录里，升级时只覆盖 exe。

**Q：怎么彻底清除所有数据（重置）？**
A：删除 `%APPDATA%\GeekClock\` 整个文件夹。

### 10.3 排错和日志

**Q：程序闪退或出错？**
A：看日志文件：exe 所在文件夹 → `logs/` 目录 → 找最新的 `.log` 文件，反馈 `[ERROR]` 内容。

**Q：日志会不会撑爆硬盘？**
A：不会。每天一个文件，自动只保留最近 7 天。

**Q：双击没反应**
A：可能性：
1. 已在运行——看托盘图标
2. 缺少 Visual C++ Runtime
3. 杀毒软件误报

## 十一、技巧推荐

### 11.1 番茄工作法

用计时器悬浮窗的倒计时模式：设 25 分钟倒计时 → 工作 → 归零响铃 → 休息 5 分钟 → 重置再来。

### 11.2 全天喝水提醒

- 名称：喝水提醒 / 分组：健康
- 间隔：60 分钟 / 活动时段：09:00 - 18:00
- 音频：选个轻柔的 / 启用渐强淡入

### 11.3 桌面少干扰模式

- 打开悬浮时钟 → 切换"小图标"模式
- 透明度调到 50%
- 拖到屏幕角落 → 锁定位置

## 十二、技术信息

- **配置文件位置**：
